🚀 Déploiement GitHub + Netlify

1. Firebase : Créez un projet sur console.firebase.google.com → Firestore (mode production) → Règles :

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

2. GitHub : Créez un repo, uploadez ce fichier HTML.

3. Netlify : "New site from Git" → sélectionnez votre repo → déployez.

4. Partagez l'URL Netlify avec votre famille — la sync est automatique.

⚠️ Pensez à mettre à jour la config Firebase en haut du script avec vos propres clés.
